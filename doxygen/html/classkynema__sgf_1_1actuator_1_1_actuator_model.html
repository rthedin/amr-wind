Detailed Description

Abstract representation of an actuator source in the flow field.

This class is intended to represent a single object (e.g., turbine, fixed wing) that exists in the computational domain. A collection of source objects (e.g., turbines in a wind farm) is modeled using the kynema_sgf::Actuator physics class that holds a collection of instances of ActuatorModel and acts on them.
