Detailed Description

template<typename LinOp>
class kynema_sgf::pde::DiffSolverIface< LinOp >

Base class for all diffusion operators

This class provides the common operations for an implicit solution of a convection-diffusion equation within Kynema-SGF.

Template Parameters
LinOpThe linear operator (see AMREeX docs)
